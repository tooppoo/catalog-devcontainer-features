# Dev Container Features

This repository publishes Dev Container Features.

## Features

- [`git-kura`](https://github.com/tooppoo/git-kura): a keyed Git worktree resolver.
  - [`src/git-kura`](https://github.com/tooppoo/catalog-devcontainer-features/tree/main/src/git-kura)
- [`rellog`](https://github.com/tooppoo/rellog): VCS-independent changelog manager
  - [`src/rellog`](https://github.com/tooppoo/catalog-devcontainer-features/tree/main/src/rellog)
- [`reportage`](https://github.com/tooppoo/reportage): an explicit, runtime-agnostic, coverage-aware E2E scenario runner.
  - [`src/reportage`](https://github.com/tooppoo/catalog-devcontainer-features/tree/main/src/reportage)

## Publishing

The release workflow publishes features from `src/` to GitHub Container Registry.
